Use the same components from plain HTML, React, Vue, Svelte, or any runtime that renders DOM.

Components ship as standards-based elements with encapsulated behavior and predictable markup.

Import the whole library for prototypes or load individual controls for tighter bundles.

Authoring and public paths are typed so component APIs stay clear in larger codebases.

Theme values flow through CSS variables, making brand colors and density easy to tune.

Server rendering helpers keep markup generation available when hydration is not the first step.
