# ReadFile

Read file content.

## API

### Return

- Promise

| Parameter | Description  | Type     | Description            |
| --------- | ------------ | -------- | ---------------------- |
| data      | File content | `string` | File content as string |

### Options

| Parameter | Description       | Type     | Default   |
| --------- | ----------------- | -------- | --------- |
| path      | File path to read | `string` | undefined |
