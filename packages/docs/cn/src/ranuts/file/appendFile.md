# AppendFile

追加一些数据到文件内

## API

### Return

- Promise

| 参数    | 说明                                 | 类型      | 描述                         |
| ------- | ------------------------------------ | --------- | ---------------------------- |
| success | 是否追加成功                         | `boolean` | true 追加成功 false 追加失败 |
| data    | 追加失败的原因，添加成功后的文件内容 | `any`     |                              |

### Options

| 参数    | 说明                     | 类型     | 默认值    |
| ------- | ------------------------ | -------- | --------- |
| path    | 文件路径，需要追加的文件 | `string` | undefined |
| content | 需要追加的内容           | `string` | 无        |
