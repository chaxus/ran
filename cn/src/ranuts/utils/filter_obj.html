filterObj ​

过滤对象的属性，去除对象中在 list 数组里面有的属性，返回一个新对象，一般是用于去除空字符和 null

API ​

Return ​

参数说明类型
Object返回的一个对象Object

Options ​

参数说明类型默认值
obj需要过滤的对象object无
list需要过滤的熟悉数组array无

Example ​

js
import { filterObj } from 'ranuts';

const obj = {
  name: 'chaxus',
  age: 10,
  address: 'spark',
};

const result = filterObj(obj, ['name', 'address']);

console.log(result);

// { age:10 }
